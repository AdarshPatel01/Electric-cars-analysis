About the project

This Electric Vehicle Data Analytics project showcases how data visualization can uncover meaningful patterns in EV adoption, charging infrastructure, and regional performance.

Explore Dashboard

Project overview

The goal is to transform raw EV-related data into a clear, decision-ready view. The dashboard supports exploration with filters, comparisons, and geographic breakdowns, while the story presents a structured narrative for key takeaways.
